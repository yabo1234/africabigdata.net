# EU–AU Educational Resource | africabigdata.net

## 🌍 Overview

This is a comprehensive educational resource documenting the relationship between the European Union and the African Union, featuring detailed coverage of Global Gateway initiatives, bilateral partnerships, AI governance resources, and institutional frameworks.

## 📁 Website Structure

### Core Pages
- **`index.html`** - Homepage with overview and navigation
- **`eu-overview.html`** - European Union institutions and policies
- **`au-overview.html`** - African Union structure and Agenda 2063
- **`partnership.html`** - EU-AU cooperation and Global Gateway (€150B investment)
- **`eu-ethiopia.html`** - Bilateral EU-Ethiopia cooperation and RISED project
- **`ai-resources.html`** - African AI governance and digital policy resources
- **`about.html`** - Portal mission and editorial information
- **`contact.html`** - Contact forms and team information
- **`privacy.html`** - Privacy policy and data protection
- **`cookies.html`** - Cookie policy and management

## 🚀 Deployment Instructions

### For Netlify (Recommended)
1. Download all files from this folder
2. Create a zip file containing all HTML files
3. Go to [netlify.com](https://netlify.com) and sign in
4. Drag and drop the zip file onto your Netlify dashboard
5. Your site will be live in minutes with automatic HTTPS

### For Other Hosting
1. Upload all HTML files to your web server root directory
2. Ensure proper MIME types are configured for HTML files
3. Set up redirects if needed for SEO
4. Configure SSL certificate for HTTPS
